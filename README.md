
# Projeto Dashboard Xbox

Esse projeto busca apresentar os conhecimentos obtidos através do bootcamp Santander 2025 - Ciência de Dados com Python.

Foi utilizada uma base de dados fictícia disponibilizada pela plataforma para o desenvolvimento do projeto. Nela, continha informações sobre o usuário como seu nome, data da compra de determinados produtos, se a pessoa escolheu ter um plano anual ou não e etc.

O desenvolvimento do projeto teve a ajuda do professor Felipe Aguiar que colaborou com o lecionamento do curso.
